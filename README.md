# Manual Monitoreo Procesos LISTO

Sitio estatico listo para desplegar en Vercel.

## Despliegue

Desde esta carpeta:

```powershell
cd C:\Automatizaciones\procesoAM\vercel-manual
npx vercel
```

Para produccion:

```powershell
npx vercel --prod
```

No desplegar la carpeta completa `procesoAM`, porque contiene reportes y evidencias locales.
