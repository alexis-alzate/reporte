Manual operativo actualizado

Automatizacion Monitoreo Procesos LISTO

Version actual: sin Power Automate. Python revisa servidores, genera evidencias, crea carpetas por fecha y copia los archivos a OneDrive para que se sincronicen con SharePoint.

Flujo Actual Instalacion OneDrive Ejecucion Resultados Errores Checklist Comandos

1. Flujo actual

El proceso ya no necesita Power Automate Desktop para copiar archivos a SharePoint.

proceso2.py
  -> consulta servidores LISTO
  -> genera Reportes_Monitoreo
  -> crea carpeta Año / Mes / Fecha / Revisión en OneDrive
  -> limpia archivos anteriores de esa revision
  -> copia archivos nuevos
  -> OneDrive sincroniza con SharePoint
Archivo principal: ejecutar proceso2.py o usar ejecutar.bat.

2. Que revisa

El script consulta:

http://172.16.11.7/ProcesosListo/procesos2.html
...
http://172.16.28.7/ProcesosListo/procesos2.html
https://listo-interno.listoonline.co/ProcesosListo/procesos2.html
https://listo24clientes.listoonline.co/ProcesosListo/procesos2.html

Busca estados de alerta:

sin va
sin es
sin po
sin so

Los casos definidos como excepciones no se reportan como novedad.

3. Instalacion por equipo

Copiar la carpeta del proyecto en:

C:\Automatizaciones\ProcesoAM

Archivos minimos:

ProcesoAM
|-- proceso2.py
|-- proceso.py
|-- ejecutar.bat
`-- requirements.txt

Instalar dependencias:

cd C:\Automatizaciones\ProcesoAM
python --version
python -m pip install -r requirements.txt
Importante: si python --version no responde, primero se debe instalar Python.

4. VPN

Los servidores 172.16.x.x son internos. Para consultarlos se requiere VPN o red corporativa.

Test-NetConnection 172.16.11.7 -Port 80
Correcto: TcpTestSucceeded : True.
Incorrecto: TcpTestSucceeded : False. Conectar VPN y repetir.

5. OneDrive y SharePoint

El script copia a la carpeta local sincronizada de OneDrive:

%USERPROFILE%\OneDrive - PLAY TECHNOLOGIES S.A.S\Área de Soporte - Monitoreo 7AM

Ejemplo:

C:\Users\Desarrollo\OneDrive - PLAY TECHNOLOGIES S.A.S\Área de Soporte - Monitoreo 7AM

Si esa carpeta existe y OneDrive esta sincronizando, lo que Python copie alli aparece en SharePoint.

No se guardan credenciales en Python: OneDrive sube usando la sesion corporativa del usuario.

6. Ejecucion

Opcion recomendada

Dar doble clic en:

C:\Automatizaciones\ProcesoAM\ejecutar.bat

El BAT pregunta si es primera o segunda revision.

Opcion PowerShell

cd C:\Automatizaciones\ProcesoAM
python proceso2.py

Para segunda revision:

python proceso2.py "segunda revisión"

8. Errores comunes

ErrorCausaSolucion
No aparece en SharePoint OneDrive no ha sincronizado o se esta mirando otro mes. Revisar la carpeta local de OneDrive y esperar sincronizacion.
Se miro Julio pero no aparece nada La fecha actual aun es junio. Buscar en 2026\06.junio\2026-06-30.
python no se reconoce Python no esta instalado o no esta en PATH. Instalar Python y validar python --version.
ModuleNotFoundError Faltan librerias. Ejecutar python -m pip install -r requirements.txt.
connection timed out No hay VPN o el servidor no responde. Validar VPN con Test-NetConnection.
