Automatizacion Monitoreo Procesos LISTO

Guia para instalar, ejecutar y validar la automatizacion de revision de procesos LISTO en equipos Windows, incluyendo VPN, evidencias locales y consideraciones de SharePoint/OneDrive.

Objetivo Instalacion VPN Ejecucion Resultados SharePoint

1. Objetivo

Esta automatizacion revisa el estado de procesos publicados por servidores LISTO, detecta novedades, guarda evidencia HTML y deja la informacion organizada para seguimiento operativo.

Entrada

Paginas HTML publicadas por servidores LISTO en /ProcesosListo/procesos2.html.

Salida

Archivos HTML por servidor dentro de Reportes_Monitoreo.

2. Que revisa

El script consulta servidores internos y externos:

172.16.11.7
172.16.12.7
172.16.13.7
...
172.16.28.7
listo-interno.listoonline.co
listo24clientes.listoonline.co

Busca estados que puedan indicar novedad:

sin va
sin es
sin po
sin so

Si el caso coincide con una excepcion definida en el codigo, no se reporta como alerta.

3. Paquete de instalacion

La USB debe llevar una carpeta llamada procesoAM con estos archivos:

procesoAM
|-- proceso.py
|-- proceso2.py
|-- ejecutar.bat
|-- requirements.txt
|-- MANUAL_INSTALACION.md
|-- MANUAL_INSTALACION.txt
`-- MANUAL_EQUIPO.html

Instalacion desde USB

  1. Crear la carpeta C:\Automatizaciones.
  2. Copiar procesoAM desde la USB hacia C:\Automatizaciones.
  3. Validar que exista C:\Automatizaciones\procesoAM\proceso.py.
  4. Instalar Python si el equipo no lo tiene.
  5. Instalar dependencias.
cd C:\Automatizaciones\procesoAM
python --version
python -m pip install -r requirements.txt

4. Validacion de VPN

Los servidores 172.16.x.x son internos. Para consultarlos se necesita VPN o red corporativa.

Test-NetConnection 172.16.11.7 -Port 80
Correcto: TcpTestSucceeded : True.
Incorrecto: TcpTestSucceeded : False. Conectar VPN y repetir la prueba.

5. Ejecucion diaria

Opcion recomendada

Dar doble clic en:

C:\Automatizaciones\procesoAM\ejecutar.bat

Opcion PowerShell

cd C:\Automatizaciones\procesoAM
python proceso.py

6. Resultados

Los resultados quedan en:

C:\Automatizaciones\procesoAM\Reportes_Monitoreo

Estructura esperada:

Reportes_Monitoreo
`-- 2026-06-25
    `-- primera revision
        |-- server 11.html
        |-- server 12.html
        |-- ...
        |-- listoaws.html
        `-- listo24clientes.html

7. SharePoint, OneDrive y Power Automate

Importante: el script Python no sube archivos directamente a SharePoint.

Si la evidencia aparece automaticamente en SharePoint, esa parte la realiza una configuracion externa de OneDrive o Power Automate.

ComponenteResponsabilidad
proceso.pyConsulta servidores y guarda archivos localmente.
OneDriveSincroniza una carpeta local con SharePoint si esta configurado.
Power AutomatePuede copiar o mover archivos hacia SharePoint mediante un flujo.

8. Errores frecuentes

ErrorCausaSolucion
can't open file C:\Users\...\proceso.py PowerShell esta en otra carpeta. Ejecutar cd C:\Automatizaciones\procesoAM.
Could not open requirements file No esta en la carpeta del proyecto. Entrar al proyecto e instalar dependencias.
connection timed out No hay VPN o el servidor no responde. Validar VPN con Test-NetConnection.
