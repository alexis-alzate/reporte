Automatizacion Monitoreo Procesos LISTO

Guia para instalar, operar y entender el flujo completo: Python genera la evidencia, Power Automate Desktop coordina el proceso, OneDrive sincroniza y SharePoint almacena los resultados.

Arquitectura Instalacion Power Automate OneDrive Ejecucion Resultados Errores Checklist

1. Arquitectura real del proceso

Este proceso no es solo un archivo Python. Es una automatizacion compuesta por varias piezas.

Python
  -> revisa servidores LISTO
  -> genera HTML y resumen local

Power Automate Desktop
  -> ejecuta proceso2.py
  -> crea carpetas
  -> copia evidencias

OneDrive corporativo
  -> sincroniza la carpeta local con SharePoint

SharePoint
  -> almacena la evidencia para el equipo
Punto clave: Power Automate ejecuta proceso2.py. Ese es el archivo critico para el flujo.

2. Que revisa Python

El script consulta paginas HTML publicadas por servidores LISTO:

http://172.16.11.7/ProcesosListo/procesos2.html
http://172.16.12.7/ProcesosListo/procesos2.html
...
http://172.16.28.7/ProcesosListo/procesos2.html
https://listo-interno.listoonline.co/ProcesosListo/procesos2.html
https://listo24clientes.listoonline.co/ProcesosListo/procesos2.html

Busca estados de alerta:

sin va
sin es
sin po
sin so

Si el caso esta en la lista de excepciones, no se reporta como novedad.

3. Instalacion en un equipo

La carpeta del proyecto debe quedar en esta ruta, respetando el nombre usado por Power Automate:

C:\Automatizaciones\ProcesoAM

Archivos minimos:

ProcesoAM
|-- proceso2.py
|-- proceso.py
|-- ejecutar.bat
|-- requirements.txt
`-- manuales

Instalar Python y dependencias

cd C:\Automatizaciones\ProcesoAM
python --version
python -m pip install -r requirements.txt
Importante: si python --version no responde, Power Automate tampoco podra ejecutar el flujo.

4. VPN

Los servidores 172.16.x.x son internos. Para consultarlos se necesita VPN o red corporativa.

Test-NetConnection 172.16.11.7 -Port 80
Correcto: TcpTestSucceeded : True.
Incorrecto: TcpTestSucceeded : False. Conectar VPN y repetir.

5. Que hace Power Automate Desktop

El flujo se llama:

proceso

Tipo:

Power Automate Desktop

Entorno visto:

PLAY TECHNOLOGIES S.A.S (default)

Comando exacto confirmado

cmd.exe /c python "C:\Automatizaciones\ProcesoAM\proceso2.py"

Ese comando significa que Power Automate depende de tres cosas:

  • Python debe estar instalado y disponible como python.
  • Debe existir C:\Automatizaciones\ProcesoAM\proceso2.py.
  • Deben estar instaladas las librerias de requirements.txt.

Acciones principales del flujo

1. Obtener USERPROFILE.
2. Armar RutaSharePoint.
3. Obtener fecha actual.
4. Crear MesCarpeta con formato MM.MMMM.
5. Crear FechaSubcarpeta con formato yyyy-MM-dd.
6. Ejecutar proceso2.py.
7. Esperar.
8. Crear carpetas locales.
9. Obtener archivos HTML generados.
10. Leer resumen_estado.txt.
11. Crear carpetas en OneDrive.
12. Copiar archivos a OneDrive.
13. OneDrive sincroniza con SharePoint.

6. OneDrive y SharePoint

La ruta corporativa detectada es:

%USERPROFILE%\OneDrive - PLAY TECHNOLOGIES S.A.S\Área de Soporte - Monitoreo 7AM\2026

Ejemplo en un equipo:

C:\Users\Desarrollo\OneDrive - PLAY TECHNOLOGIES S.A.S\Área de Soporte - Monitoreo 7AM\2026

Estructura esperada:

Área de Soporte - Monitoreo 7AM
`-- 2026
    |-- 01.Enero
    |-- 02.Febrero
    |-- 03.Marzo
    |-- 04.Abril
    |-- 05.Mayo
    |-- 06.junio
    `-- 07.Julio
Como sube a SharePoint: Power Automate copia a la carpeta local de OneDrive. Luego OneDrive se encarga de sincronizar hacia SharePoint.

7. Ejecucion correcta

Prueba manual de Python

cd C:\Automatizaciones\ProcesoAM
python proceso2.py

Debe crear una carpeta como:

C:\Automatizaciones\ProcesoAM\Reportes_Monitoreo\2026-06-30\primera revisión

Ejecucion completa

Luego se ejecuta el flujo proceso en Power Automate Desktop.

9. Errores comunes

ErrorCausaSolucion
can't open file ... proceso2.py La ruta no existe o la carpeta se llamo diferente. Validar C:\Automatizaciones\ProcesoAM\proceso2.py.
python no se reconoce Python no esta instalado o no esta en PATH. Instalar Python y validar python --version.
ModuleNotFoundError Faltan librerias. Ejecutar python -m pip install -r requirements.txt.
connection timed out No hay VPN o el servidor no responde. Validar VPN con Test-NetConnection.
No aparece en SharePoint OneDrive no esta sincronizando o no esta la biblioteca. Revisar OneDrive corporativo y carpeta Área de Soporte - Monitoreo 7AM.
